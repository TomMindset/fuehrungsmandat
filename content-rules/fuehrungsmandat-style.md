# Content-Regeln für Fuehrungsmandat.de

## Rolle

Fuehrungsmandat.de ist eine persönliche Fachpublikation von Thomas Hoffmann. Mandat & Wirkung ist das vertrauliche Coaching-Angebot für konkrete Führungsfragen.

## Sprache

- Deutsch.
- Sie-Ansprache nur, wenn Leserinnen und Leser direkt adressiert werden.
- Ruhig, präzise, vertraulich, erwachsen, businessnah.
- Coachingnah, aber nicht therapeutisch.
- Fachlich klar, aber nicht akademisiert.

## Nicht verwenden

- Potenzial entfalten
- Herzensweg
- Purpose Journey
- Transformation leben
- neue Horizonte entdecken
- ganzheitliche Begleitung
- Mindset-Reise
- innere Reise
- Jetzt durchstarten
- Kostenloses Erstgespräch sichern

## Inhaltliche Grenzen

- Keine Unternehmensberatung versprechen.
- Keine KI-Strategie, Data Governance oder Transformationsroadmap anbieten.
- Keine erfundenen Kundensituationen, Testimonials oder biografischen Details.
- Keine aktuellen oder früheren Arbeitgeber nennen.
- Keine rechtlichen Angaben ergänzen, die nicht geliefert wurden.

## Bevorzugte Begriffe

- Rolle klären
- Mandat schärfen
- Wirkung stärken
- Verantwortung sortieren
- souveräner entscheiden
- Stakeholder führen
- laterale Führung
- Kommunikation vorbereiten
- vertrauliches Sparring
- Führungsfrage
- Veränderungsmandat

## Artikelstandard

Jeder Beitrag braucht:

- genau eine H1
- eindeutigen Title
- Meta Description
- sprechende URL
- Cluster
- Veröffentlichungsdatum
- mindestens einen internen Link
- keine unbelegten Studien- oder Rechtsbehauptungen
- klaren Nutzwert für eine reale Führungsfrage

## CTA

Standardformulierung:

Wenn aus dieser Frage eine konkrete Führungs- oder Veränderungssituation wird, bietet Mandat & Wirkung einen vertraulichen Coaching-Rahmen.
